# SmileSchool Landing Page

Welcome to the **SmileSchool** landing page – a vibrant, modern, and engaging front-end project that showcases a fictional online tutorial platform where users can learn to master the art of smiling.

## 🚀 Project Overview

This project is part of the **ALX HTML & CSS curriculum**, specifically the `css_advanced` module. It includes:

* A hero section with a strong call-to-action
* Instructor profiles with images and bios
* A testimonial section with a clean, modern background
* A showcase of popular tutorial cards with ratings and duration
* A benefits section for free membership
* A frequently asked questions (FAQ) section
* Footer with social icons and branding

Built with semantic **HTML5** and Font Awesome for icons, this static page is styled for easy enhancement with CSS or a frontend framework.
